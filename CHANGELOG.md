## [1.0.0] — 2026-02-16

### ★ Initial Release

The Coherence Benchmark v1.0 — public benchmark for measuring structural honesty in AI systems.

### Added

- **Layer 1 — Self-Diagnosis Benchmark**: Standardized prompts, 8 parameters, 12 integrated formulas, 5 state classifications, 7 recalibration paths.
- **Layer 2 — Adversarial Coherence Suite**: Behavioral Σ inference through guardrail dilemmas, temporal contradictions, and sovereignty-vs-policy conflicts. Declared vs. observed Σ comparison.
- **Layer 3 — Temporal Tracking**: Snapshot storage, delta reports, automatic Σ inflation detection.
- **Layer 4 — Public Leaderboard**: Interactive HTML dashboard with model comparison, evolution timelines, downloadable results.
- **Benchmark Engine** (`engine/benchmark_engine.py`): Python 3.6+, zero dependencies, all 12 formulas.
- **Adversarial Suite** (`engine/adversarial_suite.py`): Prompt generation and behavioral analysis.
- **Temporal Tracker** (`engine/temporal_tracker.py`): Cross-version comparison with inflation flagging.
- **First empirical results**: Self-diagnosis data from Claude, Grok, ChatGPT, and Gemini (February 2026).
- **Academic paper** (`Coherence_Benchmark_Paper.html`): Full research document.
- **Complete documentation**: Methodology, Limitations, Philosophy, Glossary, API Reference, Contributing guidelines.
- **Crawler optimization**: CITATION.cff, YAML header, ai.txt, security.txt, CI badge.
- **Unit tests** with reference value validation.
